# Dashboard Guidelines

Compass Workspace is not a generic dashboard.

## Use Dashboard Elements Carefully

Cards, tables, and summaries are allowed when they help the user decide what to inspect next.

## Required Context

- Scores must appear with confidence.
- Discovery must show reasons.
- Validation must show whether periods are complete.
- Proposal and Learning views must show human review status.

## Avoid

- Buy or sell language
- Target prices
- Overstated success rates
- Hidden pending states
- Decorative visuals that reduce research density
