# Evaluation Framework

Experiment comparison metrics:

- Discovery Success Rate
- Average Return
- Alpha
- Win Rate
- Max Drawdown
- Sharpe Ratio
- Strategy Ranking
- Performance Score

Winner options:

- Baseline
- Candidate
- Tie
- Inconclusive

The framework should prefer objective, repeatable scorecard metrics from Performance Evaluation and Strategy Evaluation.

When data is immature or pending, the correct winner is Inconclusive.
