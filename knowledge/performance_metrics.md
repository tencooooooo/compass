# Performance Metrics

Performance Evaluation is Compass's scorecard.

Core metrics:

- Discovery Success Rate
- Average Return
- Median Return
- Win Rate
- Loss Rate
- Alpha vs Benchmark
- Max Drawdown
- Average Holding Return

Standard periods:

- 30 days
- 90 days
- 180 days
- 365 days

Rows should remain pending until the full evaluation period has elapsed and price data is available.
