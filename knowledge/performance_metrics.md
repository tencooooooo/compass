# Performance Metrics

Validation結果から将来利用する評価指標を整理します。

## Current Metrics

- Discovery Score
- Return
- Benchmark Difference
- Sector Difference
- Validation Result
- Confidence

## Future Metrics

### Hit Rate

Good以上になった候補の割合。

### Average Return

検証対象全体の平均騰落率。

### Max Drawdown

候補が検証期間中にどれだけ下落したかを測る指標。将来追加予定。

### Sharpe Ratio

リスク調整後リターン。将来、日次リターンを用いて検討する。

### Win/Loss Ratio

良好だった候補と悪かった候補の比率。

### Confidence Accuracy

High, Medium, LowのConfidenceが実際の結果とどの程度一致したかを検証する。

## Principle

Compassは短期リターンだけで評価しません。理由、根拠、データ品質、セクター環境を含めて検証します。
