# Graph Schema

Storage:

```text
storage/knowledge_graph/graph.json
storage/knowledge_graph/nodes.json
storage/knowledge_graph/edges.json
```

Node fields:

```text
id
type
label
properties
```

Edge fields:

```text
source
target
relationship
weight
evidence
```

Dashboard graph data is also exported to:

```text
reports/graph/graph.json
```
