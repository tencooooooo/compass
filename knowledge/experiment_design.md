# Experiment Design

Compass experiments compare a baseline version with a candidate version.

Experiment fields:

- Experiment ID
- Name
- Description
- Target
- Baseline Version
- Candidate Version
- Start Date
- End Date
- Status

Initial experiment targets:

- Discovery Rule
- Scoring Rule
- Theme Rule
- Pattern Rule
- Learning Rule
- Data Source

Experiments should be reproducible from configuration, stored reports, and memory.
