# Search Strategy

Compass search should be centralized and explainable.

## Search Targets

- Companies
- Scores
- Discovery
- Validation
- Market
- Sectors
- Memory
- Feedback
- Learning
- Notifications

## Strategy

- Start with named queries for stable behavior.
- Prefer API-facing services over raw file access.
- Use Memory for historical company context.
- Use reports for human-readable Markdown summaries.
- Keep results structured so Workspace, MCP, Chat, Slack, and Mobile can render them consistently.

## Future Direction

Future search can add:

- Keyword search
- Time range filters
- Sector filters
- Confidence filters
- Full-text report search
- Vector search

These should extend Query Engine rather than bypass it.
