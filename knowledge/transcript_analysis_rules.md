# Transcript Analysis Rules

Transcript collection rules:

- Keep raw transcript paragraphs intact.
- Store metadata separately from transcript text.
- Extract only basic structural sections in the collection layer.
- Do not summarize, score, or judge management tone during collection.
- Validate metadata before saving.
- Detect duplicate transcripts by ticker, fiscal quarter, earnings date, and source.
- Keep index entries sorted by earnings date.

Basic sections:

- Opening Remarks
- Financial Highlights
- Guidance
- Q&A Section
- Closing Remarks

Future AI analysis should cite transcript paragraphs and preserve traceability to the original source.
