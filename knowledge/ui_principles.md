# UI Principles

Compass UI should feel calm, dense, and research-focused.

## Principles

- Prioritize scanability over decoration.
- Keep read-only states clear.
- Use tables for comparison and Markdown for source reports.
- Show uncertainty, missing data, and pending review status.
- Support dark mode and responsive layouts.
- Avoid turning Compass into a prediction or trading interface.

## Navigation

Navigation should reflect the research workflow:

```text
Home
Discovery
Company
Comparison
Validation
Proposal
Learning
Settings
```
