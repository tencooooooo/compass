# Portfolio Metrics

Strategy Evaluation calculates:

- CAGR
- Total Return
- Win Rate
- Sharpe Ratio
- Max Drawdown
- Alpha
- Beta
- Volatility
- Average Holding Period

Metrics should be treated as research diagnostics.

If benchmark price data is unavailable, benchmark-dependent metrics must remain `N/A` instead of being estimated.

Portfolio simulation starts from a default initial capital of 100000 USD and uses equal-weight allocation unless the strategy configuration says otherwise.
