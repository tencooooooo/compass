# Data Sources

Compass accuracy depends on data quality before model complexity.

Planned expansion categories:

- Macro: CPI, FOMC, employment, interest rates, VIX
- SEC: 10-K, 10-Q, 8-K
- Earnings: earnings presentations, guidance
- Analyst: consensus, EPS estimates
- Insider: insider buying, insider selling
- ETF: fund flows, holdings weight
- Sentiment: Reddit, X, news sentiment
- Trends: Google Trends
- Jobs: hiring trends and job postings

Current state:

- Collector scaffolds exist.
- External API connections are not implemented.
- API keys and credentials are not required.
- Collectors are disabled until a source is reviewed and approved.

Future source onboarding should document licensing, update frequency, field definitions, rate limits, retention, and quality risks before production use.
