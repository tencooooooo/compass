# MCP Usage Examples

Example requests for MCP clients:

```text
Analyze NVDA
Top Discovery
Market Summary
Validation Summary
Show NVDA History
```

## Expected Behavior

`Analyze NVDA` returns:

- Company report
- Latest Score
- Discovery context
- Validation context

`Show NVDA History` returns:

- Company Memory
- History
- Learning context
- Feedback context

## Client Guidance

MCP clients should ask Compass through MCP tools and resources. They should not parse local Markdown, JSON, CSV, or Memory files directly.
