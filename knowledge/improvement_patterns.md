# Improvement Patterns

Feedback Engineが検出する改善候補の整理場所です。

## Candidate Areas

- Scoring weight review
- Confidence calibration
- Sector-specific adjustment
- Event and news classification review
- Momentum horizon review
- Valuation sensitivity review
- Data completeness review

## Review Rule

改善候補はKnowledgeの自動更新ではありません。

人間が根拠、データ量、Validation完了状況を確認してからKnowledgeへ反映します。
