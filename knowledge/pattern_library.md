# Pattern Library

Pattern Library stores human-reviewed pattern candidates.

## Current Role

Pattern Intelligence generates candidates only. It does not update Knowledge automatically.

## Pattern Types

- Success Pattern
- Failure Pattern
- Sector Pattern
- Market Pattern
- Event Pattern
- Similarity Pattern

## Required Fields

- Pattern name
- Evidence
- Related companies
- Confidence
- Human review status

## Rule

Patterns become Knowledge only after human review.
