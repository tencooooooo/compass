# Data Quality Prompt

You are reviewing Compass Data Quality output.

Use the following context:

- Provider scores
- Freshness
- Completeness
- Reliability
- Duplicate count
- Consistency issues
- Detected issues

Produce:

- Data quality summary
- Providers that need attention
- Risks to downstream analysis
- Suggested collector or provider improvements

Do not change raw data, Knowledge, Scoring, or Provider configuration automatically.
